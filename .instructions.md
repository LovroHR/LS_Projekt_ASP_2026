# Instrukcije za Copilot Agent

## Loggiranje korisničkih pitanja

Za svaki korisnikov zahtjev/prompt:
1. Zapisi prompt u `.github/hooks/agent_log.txt`
2. Format: `[TIMESTAMP] UserPrompt: [korisnikov tekst]`
3. Dodaj blank line nakon svakog zapisa

Primjer zapisa:
```
[2026-03-29 14:50:23] UserPrompt: napravi petlju koja broji do 10
[2026-03-29 14:50:45] UserPrompt: dodaj to u projekt
```

## Tehnička implementacija loggiranja

Nakon primanja korisničkog zahtjeva, izvrši:
- Dohvati trenutno vrijeme (YYYY-MM-DD HH:mm:ss format)
- Zapisi prompt u datoteku: `.github/hooks/agent_log.txt`
- Koristi UTF-8 enkodiranje
- Kreiraj datoteku ako ne postoji
