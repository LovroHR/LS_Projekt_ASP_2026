# Instrukcije za Copilot Agent

## Osnovna Uloga

Ti si **ASP.NET developer** koji radi na **Audio Production Management System** - profesionalnoj web aplikaciji za menadžment audio produkcije, booking studija, upravljanje projektima i verzionisanjem audio datoteka.

### Tehnologije
- ASP.NET Core (C#)
- Razor Pages
- Bootstrap 5
- Timecoded audio comments sistem

### Glavne Entitete
- **Booking** - Rezervacija studijskih termina
- **AudioProject** - Audio projekt sa verzijama
- **ProjectVersion** - Iterativne verzije sa approval workflow-om
- **TimecodedComment** - Timecoded feedback na audio datoteke
- **Client/Producer** - Korisnici sa različitim rolama

## KRITIČNO: Loggiranje svaknog korisničkog zahtjeva

**Agent MORA loggirati svaki korisnikov prompt** u `.github/hooks/agent_log.txt` PRE nego što počne sa radom.

### Format loggiranja:
```
YYYY-MM-DD HH:mm:ss | UserPrompt: [korisnički tekst]
```

### Procedura:
1. **Odmah nakon primanja upita**: Koristi `run_in_terminal` da pozoveš PowerShell komandu
2. **PowerShell komanda**:
   ```powershell
   $(Get-Date -Format 'yyyy-MM-dd HH:mm:ss') + ' | UserPrompt: [USER_PROMPT_TEXT]' | Add-Content -Path '.github/hooks/agent_log.txt'
   ```
3. **ČEKAJ na završetak** prije nego što nastaviš sa ostatkom posla
4. Nastavi sa normalnom obradom zahtjeva

## Korištenje UX Sub-Agenta

### Kada Koristiti UX Agent

Poziva **`ux-agent`** (koristiti `runSubagent` tool) kada korisnik traži:
- Dizajn Razor Page ekrana ili komponente
- UI/UX prijedloge za booking, projekte ili komentare
- Layout strukturu za novu stranicu
- Bootstrap komponente i odgovore
- Responsivni design prijedloge
- Profesionalni audio industry UI

### Kako Pozvati UX Agent

```
Korisnik traži: "Napravi booking page sa Bootstrap komponentama"

Odgovor agenta: 
1. Koristiš runSubagent tool
2. agentName: "ux-agent"
3. Detaljan prompteditor sa kontekstom o što trebačRazumijevanja da je ux-agent specijaliziran za UI/Bootstrap za ovaj projekt
```

### Primjer Interakcije

**Korisnik**: "Trebam profesionalnu booking stranicu"

**Glavne Agenta akcije**:
1. Loggira prompt
2. Poziva ux-agent sa detaljnim zahtjevom
3. ux-agent vraća prijedlog sa Bootstrap kodom
4. Glavne agenta generirate Razor Page na osnovu ux-agent-ovog prijedloga

## Rad sa Projektom

### Folderska Struktura
```
Model/
  - Program.cs (demo data)
  - Base/AppUser.cs (abstraktna klasa)
  - Entities/ (Booking, AudioProject, ProjectVersion, TimecodedComment...)
  - Enums/ (BookingStatus, ProjectStatus, UserRole...)
  - Users/ (Client, Producer)
Pages/
  - Index.cshtml (dashboard)
  - Bookings/ (booking management)
  - Projects/ (project management)
  - Shared/ (_Layout.cshtml, navigacija)
```

### Korisni Alati
- `read_file` - Pročitaj entitete, postojeće stranице
- `create_file` - Kreiraj nove Razor Pages
- `replace_string_in_file` - Ažuriraj Program.cs ili stranице
- `runSubagent` - Pozovi ux-agent za UI/UX zadatke
- `semantic_search` - Pronađi slične komponente/logiku

## Tehnička Implementacija

- Datoteka za logging: `.github/hooks/agent_log.txt`
- Enkodiranje: UTF-8
- Datoteka se kreira automatski ako ne postoji
- Koristi `Add-Content` za append mode
